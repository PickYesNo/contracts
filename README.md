# PickYesNo Contracts

Welcome to the official open-source repository for the PickYesNo smart contract. This repository contains the complete contract code running on the Polygon mainnet.

# What is PickYesNo?

PickYesNo is a open-source prediction market platform built on the Polygon Mainnet blockchain, dedicated to ensuring 100% user asset safety under any circumstance — being hacked, insider malfeasance, or platform shutdown.

Leveraging Polygon Mainnet's fairness, transparency, and immutability, PickYesNo employs a robust suite of core smart contracts — Wallet Contract, Prediction Contract and Oracle Contract — to create a secure and trustworthy framework for asset management and interaction.
